HTML Sharing Service

HTML を、
URL に。

自己完結した HTML ファイルをドラッグするだけで、
すぐに共有できる URL を発行します。

⚡ 即時発行
📦 最大 10MB
🔒 認証付き管理
🌙 ダークモード

How it works

3ステップで共有

複雑な設定は不要。ファイルを選んでドロップするだけです。

📄

Step 1

HTML を用意

自己完結した .html ファイルをひとつ用意するだけ。

⚡

Step 2

ドロップして送信

pagebox にログインして、ファイルをドラッグ＆ドロップ。

🔗

Step 3

URL を共有

発行された URL をコピーして誰にでも送れます。ログイン不要で閲覧可能。

Features

シンプルに、強く。

必要な機能だけを、丁寧に。

🔓
ログイン不要で閲覧
共有 URL はアカウント不要。受け取った人はすぐ開けます。
🚀
Cloudflare Workers
エッジで動作。世界中どこからでも高速アクセス。
🗂️
マイスペース管理
アップロード済みファイルを一覧で管理・削除できます。
🌙
ダークモード対応
OS 設定に自動追従。手動切替ボタンも完備。
